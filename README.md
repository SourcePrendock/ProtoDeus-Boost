# ProtoDeus-Boost
ProtoDeus Boost 🛠️ es un optimizador y diagnosticador premium para Windows 10/11 en Python + CustomTkinter. Ofrece limpieza profunda de temporales 🧹, optimización de RAM/SSD/Red ⚡, monitoreo de hardware en tiempo real 🖥️ y auditoría de licencias 🔑. Todo en un único .exe seguro con elevación UAC automática y una interfaz Dark Mode moderna.
